### Important HTTP Status Code
- 1.xx -> Informational
- 2.xx -> Success

- 200 -> success
- 201 -> Created and Successfull
- 204 -> No Content (when deleting)

- 300 -> redirection
- 304 -> Not modified

- 4.xx -> Client Error
- 400 -> Bad Request (give what was needed)
- 401 -> Unauthorized
- 404 -> Non Found

- 5.xx -> Server error
- 500 -> Internal Server Error
